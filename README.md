# AI Regulatory Assistant

An intelligent web application that automates regulatory deficiency responses using Amazon Bedrock.

## Features
- Upload deficiency documents
- AI-generated regulatory responses
- Structured output for health authority communication
- PDF export
- Mock fallback for reliability

## Tech Stack
- Python
- Flask
- Amazon Bedrock
- Boto3

## Setup

1. Clone the repo
2. Install dependencies:

pip install -r requirements.txt

3. Configure AWS credentials:

aws configure

4. Run the app:

python app.py
